fmri_atlas
==========

.. currentmodule:: fmri_toolbox

Description
-----------
FMRI_ATLAS  Lists all available atlases and their region counts.

Usage
-----
.. code-block:: matlab

    fmri_atlas()     % print full atlas list
    fmri_atlas(n)    % print region list for atlas n
